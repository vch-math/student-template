# Вычислительная математика — шаблон для студентов

Этот репозиторий создан через GitHub Classroom.
Решение каждой лабораторной кладите в соответствующую папку `labs/lab-<n>`.

## Как сдавать
1. Создайте ветку под лабораторную, например `lab-1`.
2. Положите решение в `labs/lab-1/` (см. README внутри).
3. Сделайте коммит и запушьте — автопроверка запустится сама.
4. Коммиты просьба создавать тоже правильно `feat: что-было сделано/исправлено`
## Поддерживаемые языки
Файл решения должен называться одним из:
- Python: `main.py`
- C: `main.c`
- C++: `main.cpp`
- Java: `Main.java`

В случае изменения шаблона мной будет создан PR‑обновление:
- У вас появится новый PR
- Вы просто мерджите PR
- Подгружаете его к себе в ветку через git pull на main ветку
- И потом переключаетесь на свою ветку lab-N
- И делаете git rebase main
